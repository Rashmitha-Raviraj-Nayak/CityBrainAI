# Hackathon Submission Checklist

## Deployment
- [ ] Firebase project is created and configured.
- [ ] Authentication providers are enabled.
- [ ] Firestore rules are deployed.
- [ ] Storage rules are deployed.
- [ ] Firestore indexes are deployed.
- [ ] Backend URL is configured for the frontend.
- [ ] Production environment variables are set.

## App Functionality
- [ ] Landing page loads correctly.
- [ ] Login and Google sign-in work.
- [ ] Incident reporting works.
- [ ] AI runtime returns structured output.
- [ ] Explainability screen renders clearly.
- [ ] Dashboard and analytics load.
- [ ] Admin views are accessible for authorized users.

## Quality Gates
- [ ] Frontend lint passes.
- [ ] Frontend typecheck passes.
- [ ] Frontend build passes.
- [ ] Backend tests pass.
- [ ] No broken routes remain.
- [ ] No critical console errors are present.

## Demo Readiness
- [ ] Demo flow is smooth and short.
- [ ] Screens look polished on desktop and mobile.
- [ ] Error states and loading states are acceptable.
- [ ] The app can be presented without manual fixes.
